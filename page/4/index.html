反思

想来大一参加社团其实收获寥寥，因为目标不明确，后来意识到自己的社交技能树以前没点过，所以就都退了。迷茫这个词年轻的时候可以任性的用一用，大二，大三，大四总不好再用这种借口开脱。而且轻言放弃是个很蠢的事情，从最后的结果上看，坚持不断学习的发哥不出意外的成功保研了，而我可不记得他当时成绩比我好很多。大二确定毕业就去工作，为了这个做了一点点准备，也确实目的清晰的做了那么一点微小的工作，但是做得并不好。我这人有种得过且过的懒人心态，再不像小时候那么争强好胜，可是实际上在小的团队里，成绩中上的混子是没有出头之日的。以后工作也是，小团队里做到最好才有价值，我要切记这一点。大三实习，非要玩漂移，既定路线走的不舒服，回头就想换方向，可是哪有那么容易，考研准备的时间不充分，还记得一夫在我三月份找实习的时候就开始准备复习了，而我九月接近十月才开始专心复习。蜜汁自信的最后结果却是求而不得，而且还是考试这种形式。除了时间不充分以外，重点不突出，而且给自己定的目标也不够高，就想着过了分数线就行。。。我确实过了往年分数线，谁知道他会提升啊。好在最后结果也不算不能接受，换个环境对我来说应该是更好了。

矫情

我说感谢室友不杀之恩固然是玩笑之语，但我的感情是认真的。虽然从没和谁发生什么矛盾争执，但我总觉得自己干了不少蠢事。因为我深知自己情商很低，除了极少情况下我索性装傻以外，大部分时候我是真傻的。必须感谢各位对我的包容。鸫哥性子很直的，也讲给我很多道理，虽然我迷迷糊糊的，还是希望能学到那么一点点，也希望鸫哥效率依旧，毕竟在我看来，鸫哥要是真心想做成一件事情，那就没什么能阻挡的住，一定要和鸫嫂幸福；发哥是真的很励志的，希望我们开黑的时候没对你造成太多干扰，也希望发哥一如既往的认真学习，不说功成名就，回报与收获总是会有的，女朋友也是会有的，而且是漂亮的。主席生活多姿多彩的很，让我这个宅属性的凡人见识了很多新操作，可是主席的能力我羡慕不来，惟愿主席立业成家一切顺利，尤其是和学姐照顾好两只金毛。而微微，冠神，虽然不知道为什么，总是觉得很高端的样子，只希望微微记得出国之后，苟富贵，勿相忘啊。隆隆，性子稳的很，有个什么事情我还是觉得拜托你很靠谱，倒是希望你和发哥学习一下，稍稍努力一点，把肚皮减一减。而天泽又高又富又帅，不嫌我菜，不嫌我贫，一时间还想不到能祝人生赢家什么，就祝你永远年轻，永远是个宝宝吧。至于翔哥，和你赌了一顿饭，虽然你总是骗我，但我真希望最后是我请你，也愿你接下来的学习和工作一切顺利。

回家

这两天回到家，身心放松睡的深沉，每次都感觉是从万丈深渊里醒来。下午收拾了一下自己黑历史，除了各种成绩单证书和照片，甚至还有小升初考完试的日记，想想也是感谢老妈。里面甚至还有高中读书每次往家里打电话之后老妈的记录，我根本不敢看，我之所以没有顺着懒散的性子堕落下去，除了机缘巧合活在一个强者如云的环境里，老妈的关心和帮助居功至伟。我其实对婚姻和孩子很恐惧，因为我知道我根本不可能像父母爱我一样爱我的孩子更不可能欺骗自己去爱自己以外的其他人。昨天去见了一下奶奶，只觉得生命脆弱，时间无情，以前总是想什么人生意义，老爸这两天倒是不断跟我分享什么是幸福：身体健康，钱够花。可我只觉得自己可怜，从不会为自己的经历热泪盈眶，我在这回顾过去也是冷静的很，往往变成了批评与自我批评，偶有感时伤怀也被自己定义为矫情。

但愿

新的旅程，我但愿自己能在选定的方向上坚持不懈地付出努力，顺便找到一个好奇不减的爱好就更好了。再大一点空一点，我但愿这世界仍是和平的主旋律，种族宗教和分裂问题不要发生在身边。我但愿中国这谜一般坚挺的房价和gdp能依旧坚挺，阶级固化不要那么快，让我们还有一丝实现阶级跨越的希望。我但愿我认识和认识我的人长命百岁，倒不是为了苟活于世，只为了见识这世界飞速发展带来的无限可能，若是没有老的糊涂，顺便看看当年身在山中不知始终的选择究竟路向何方就更好了，虽是不曾指点江山挥斥方遒，但总可以看看同一时代到底几人射雕几人比高，鼓个掌也是好的。

我发此宏愿，若是天才人物说不定就天地共鸣，异象纷程了，而我看了一眼窗外，一片漆黑，十一点了，也不知道敲了多少错别字，睡觉睡觉。

def simple(request):
    import random
    import django
    import datetime

    from matplotlib.backends.backend_agg import FigureCanvasAgg as FigureCanvas
    from matplotlib.figure import Figure
    from matplotlib.dates import DateFormatter

    fig=Figure()
    ax=fig.add_subplot(111)
    x=[]
    y=[]
    now=datetime.datetime.now()
    delta=datetime.timedelta(days=1)
    for i in range(10):
        x.append(now)
        now+=delta
        y.append(random.randint(0, 1000))
    ax.plot_date(x, y, '-')
    ax.xaxis.set_major_formatter(DateFormatter('%Y-%m-%d'))
    fig.autofmt_xdate()
    canvas=FigureCanvas(fig)
    response=django.http.HttpResponse(content_type='image/png')
    canvas.print_png(response)
    return response

百无一用是书生，读书其实没什么价值，有价值的是读书这个行动本身。这篇四不像的记录的核心价值也在于记录本身，至于里面的内容收获对于我来说其实没太大价值，于将来的我说不定略有价值，但是估计价值也不大。但我需要坚持做这件内容没什么价值的事情，因为我认为这个行动是有价值的。起码也先做一段时间再说，反正总比闲着打游戏要好，既然我己经花了有两个小时看了电影，也不必计较再花十几分钟或者再多一点记一下当时的想法吧。看一本书花的时间更多，就更是如此了。但我总要谨记的是，这里面其实真没什么有价值的内容，以后的自己也不必太认真的看。记录这东西确实花了我一些精力，但这不是收获，更不算是锦衣夜行，这里面的东西对我都没什么价值，对别人更是，只是记录癖的浪费时间。

Introduction

Machine Learning

  • The field of study that gives computers the ability to learn without being explicitly programmed.

  • A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E.

  • Supervised Learning

  • Unsupervised Learning
  • Reinforcement learning
  • Recommender systems

根据训练数据是否有标记信息，可大致把学习任务分成两类，监督学习和无监督学习。前者主要包括分类和回归。后者主要有聚类。
