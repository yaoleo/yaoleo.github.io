    captChar=""        
    if(response.url==get_url):            
        soup = BeautifulSoup(response.body, 'lxml')            
        _xsrf = soup.find("input", {'type': 'hidden', 'name': '_xsrf'}).attrs['value']        
    else:            
        with open('capt.gif', 'wb') as f:                
            f.write(response.body)            
        captChar = input('please read the image capt.gif and input the captchar:')        
    yield scrapy.FormRequest(login_url,headers=headers,                                   formdata={'_xsrf':_xsrf, 'captcha': captChar,"password": passowrd,"email": email},                                   callback=self.logged_in)    
###logged_in的处理函数，主要是获取是否成功登陆的信息，成功登陆后，发起一个request    
def logged_in(self,response):
    soup = BeautifulSoup(response.body, 'lxml')        
    login_result = eval(soup.p.string)        
    result=login_result["msg"]        
    yield {'login':result}        
    if (login_result["r"] == 0):            
        yield scrapy.Request(url=firstpage, headers=headers,        callback=self.parse_data)

    soup = BeautifulSoup(response.body, 'lxml')

Machine Learning

  • The field of study that gives computers the ability to learn without being explicitly programmed.

  • A computer program is said to learn from experience E with respect to some class of tasks T and performance measure P, if its performance at tasks in T, as measured by P, improves with experience E.

  • Supervised Learning

  • Unsupervised Learning
  • Reinforcement learning
  • Recommender systems

根据训练数据是否有标记信息，可大致把学习任务分成两类，监督学习和无监督学习。前者主要包括分类和回归。后者主要有聚类。

Supervised Learning

  • classification
  • regression

Unsupervised Learning

  • clustering 聚类

需要注意的是，机器学习的目标是学得的模型可以很好的适用“新样本”，而不仅是在训练样本上工作良好，即使对于聚类这样的无监督学习任务，也希望他能适用于新样本，称为 泛化 generaliazation

表扬与自我表扬

哈哈，越来越不要脸，给自己写记录能写出这个标题，主要是看上个月以及的记录，越来越觉得自己不够乐观和积极。这固然是性格所致，但既然意识到了，就要往好的方向强行扭一下。

首先是MEB顺利完成，收获一般吧，还好。然后烘焙事业有所进步，戚风，蛋糕卷都算比较成功。实验室和宿舍都非常满意，也没什么更高的要求了，我所需要的基本都有了。为人处世没什么进步，但是也没出什么问题，已经表现不错了。早上还能起来混个早餐，是最满意的事情，没有之一。对很多事情有了更深刻的想法和理解，也是非常棒的。

计划

首先，接下来的两周左右的时间，我一定要把Coursera上完。除了偶尔烤个蛋糕犒劳自己，每天完成任务或者计划的情况下，应该画一颗树，算是重拾画画这项计划吧。当然，这也源于我对坚持这个定义的认识丰富。然后，在做人做事上，我倒是注意到一个新概念，人格魅力。积极进取不怕失败都是老生常谈，成为团队核心是一个很重要的点。可惜我暂时不知道怎么弄。

关于读书与看剧

没读完完整的一本，找了本希腊棺材之谜，说是古典推理巅峰之作，但是不好看，可能是翻译的真的垃圾吧。而且我对图书馆的距离非常不满意，有点远，就不好制定没事看杂书的计划了。那些快速翻的书真的不适合看电子版，翻页体验太差了，根本不能实现快速浏览的效果。头疼。前几天看到俞敏洪大佬2017上半年的读书报告，大概有五十几本吧，按他的说法，基本都是在路上看的。好几本都是旅游之前看的相关国家的文化背景书。可见我走马观花的瞎逛还是和大佬差距很大的，而且看了一下自己的统计，勉强能算读了三十本，考虑时间和忙碌程度的对比，差距还是有，不过也算不错吧。重点是收获了什么，数量毫无参考价值。从读书笔记的质量上看，大佬的学识基础和思考角度不知比我高到哪里去了。

剧到是看完了权力的游戏第七季，第七季了！倒数第二集的时间线发展太快了不是很喜欢。提起时间，剧集里的二丫小姑娘已经成长到复仇者了，私生子和侏儒也都成了北境之王和国王之手了，就连被卖给异族当老婆的眉毛怪233都变身龙母反攻维斯特洛大陆了。我的成长呢？七年前我在干嘛？现在的我又在干嘛？但要说自我感觉性质的成长，那么从去年开始做记录以来倒是最大的。可能因为记忆力太差，没有记录的那部分成长都被我自己忘掉了吧。

动漫发现了一部神剧，我突然发现动漫不像电影一样受限于工业特效水平，往往可以比电影更提前丰富的表现人类的想象力，功壳机动队就是最好的例子。牛姨的这部银之匙没有任何夸张地想象力，但它真的触及了我的灵魂。我真切的感觉到自己被剧中人物的经历触动并莫名其妙的为自己流下泪水。尽管我稍稍不喜欢强行借角色之口直白的讲道理的片段，但是或许没了这个，我自己的理解能力也不能注意到那些道理。要是满分十分，我真愿意给它打12分。

关于远虑和近忧

人无远虑必有近忧。我这个人一直没什么理想和追求，这也是银之匙触动我的原因。我仔细想了一下，其实是被外界宣传的理想追求定义所限制了，我其实是有的，但是我总觉得那和改变世界，当科学家，创业，为国为民，环游世界….不能等价而论，不能算是理想。但其实不是，目标高远只是一方面，个人的喜好爱憎是另一方面。一个人说他的理想是当一个家庭主妇，是一生平安，要比那些理想是当国家主席的人有智慧的多。因为他开始跳出普世价值来真正思考自己生命的价值和意义了。起码他不会太在意外在的、无关紧要的看法了。

所以我又没有什么理想抱负呢？我原来一直以为没有，但可能也是有的：

我真的希望健康的长命百岁，要是有了疾病和痛苦，苟且偷生就算了吧。
我还希望能去意大利把教堂画下来，这是我目前能想到的B格最高的事情了，算是我个人自我实现需求里的顶点把。
我还希望自己不用为没钱看病，没钱买饭而担忧。日常购物不考虑奢侈品的话基本不需要考虑价格。

看，我这个人真是tm的自私啊。且不说家国天下，我的理想里真的没考虑其他人的存在。一个人生而孤独，考虑别人的时候只能是说顺便而为。

画教堂这件事要在五年之后，因为一个菜鸡总要花一点时间学习画画，而且我很懒还很忙,如果那时候我打破了注孤生的魔法，我还希望能和她一起摆脱那无聊的向周围熟悉的陌生人宣布关系的婚礼，而是一座一座城市的风景看过去。最好是一座一座教堂看过去，捡一个最漂亮的交换戒指，把座右铭改成I pray to ourselves, for ourselves.

努力赚钱是生活必须，我自然可以赡养父母，照顾家庭。虽然我对把无辜的生命带到这个世界受苦这件传宗接代的上帝玩笑持怀疑态度，若是躲不过去，我自然也能尽责的照顾好他。所谓达则兼济天下，若是运气更好那么一点，有那么一点点的经济自由和时间自由，我也不指望留什么有价值的东西带到坟墓里，我也不觉得疾病和贫穷可以依靠外力获得拯救，但是我肯定可以做那么一点微小的贡献的。

作为一个年轻人，我肯定会低估健康的长命百岁这件事的难度，它是一件太长期的事情了，他的难度要比拯救世界还要难的多（毕竟按个按钮就行）。坚持锻炼这种说法，连我自己都不能欺骗了。但是我对坚持也早有了新的看法，所以其实现在对我来说坚持到其实不是很难。那么难点在哪？在于没有反馈。做一件没有任何成就感的事情真的于我太难了。最关键的是要给自己设定目标。下个月我要把体重增到130，无论如何先选一个看起来最简单的吧，尽管好像和锻炼没什么关系。

中产阶级好不容易摆脱了日复一日的重复劳动，反而开始日复一日的举铁跑步自我折磨，除了为了健康和装B的自我实现，我们竟然会从自我折磨中获得精神的愉悦，这真是一件很搞笑的事情。我接下来会刻意体会一下这种愉悦感，以前还真没这样想过，佛说里的正见好像就是这样一种方法。非常有趣。

赚钱这件事情说难真的很难，对一些人来说确实又看起来简单的很。那么与我来说最简单的方式就是找个好工作，按部就班的当个高级程序员。完成我理想里的那种状态应该不是很难。那么我接下来要做什么呢？我心里清楚的很。 （未完 9.9）
