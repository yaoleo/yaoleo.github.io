批评与自我批评

这个文档倒是八月一号就建了，但是今天已经是八月十三号了，我才开始写具体内容。感觉生活状态不是很好，由此也是可见一斑。
首先，学习不应该在宿舍进行，这一点我早就有了结论，但是新宿舍真是条件太好了，上床下桌，空调吹着，完全不想背着几公斤重的电脑来回奔波。但对我这种自制力水平的人来说，显然都是幻觉。
再次，垃圾的综艺节目开始浪费我的生命。这到可能是上一个原因的后果之一，而计划看的书进度基本停滞。英语训练计划时断时续，单词还是没有背，整个学习处于一种无计划推着走的状态。
最后，没有专注的做什么事情，也没有目的清晰的做什么事情，社交水平没有提升，为人处世也没什么收获。

但这自我批评不是目的，我得把解决方法和执行反馈写下来。233我发现不论情况如何，我都能不要脸的在每个月的总结里把自己批评的体无完肤，然后下个月继续批评，很少有那么哪个月，我能对自己的表现满意。

感觉大家都变成成年人了，只有我还是一个小孩子思维。

Meb

Meb对我来说核心应该是提升听力能力和写作能力？不，可能写作能力也不会有实质的提高。英语交流能力？嗯，其实也不会有实质的提高。能实打实提高并且有用的唯有听力。可惜，进步还是一般。可能我对进步这件事想的太容易了。听力大抵到什么标准才能算i是合格呢？裸看剧，裸听Ted无难度，大概才算满足我的要求。可是我原本的水平太低了，而且达到这个标准不是仅仅听力的问题，单词和语法都不达标。说起进步，从第一次考试的13/30份到后来的18/30分，其实进步还是有的，倒也不能全面否定自己。可能完全是单方面高估了自己进步的速度吧。

Coursera

上到第四周，进度真的是慢死了，核心问题在于没有计划，错略的时间安排没有执行。

Read & Movie

电影看了一点，和衣服一起玩的时候跟着看了死侍和沉默的羔羊，记忆中好像还看了几部经典，比如看了后半段的乱世佳人，倒是在这里的电影院和同学一起看了建军大业。书其实由于保持上个月的惯性也是看了几本（如果一部中短篇小说也能算一本书的话），大概都记载2017书和电影的记录报告里了，也没什么可说的。

Baking

想法很多，我都忘了。开始有烤箱烤面包，除了考了几次鸡腿，只考过一次蛋挞，其实效果不错，个人倒是很满意，但是到现在为止还是没有面粉和面板。计划靠戚风蛋糕却遥遥无期。尽快吧。

Plan

不知道别人怎么样，救我个人来说，我做计划很少有什么真正执行成果的时候，但少有的几次按计划执行，我个人进步都会非常大，一方面我的智商正常，一方面我自我反思和理解能力其实是优秀的，而且自我期望比较高，计划就会比较严和辛苦。后来我以为计划难以完成是因为太严格了太辛苦所以才很难完成，但是事实可能并不是这样的。

就我个人的成功经验来讲，我要是能成功执行计划–长期的，规律的，辛苦的，不情不愿的事情（就目前而言，我愿意的，不消耗精力的，可以长期坚持完成的事情只有吃饭和睡觉），那么必然需要时间和空间的独立和仪式感。

虽然每天放学都在一直玩然后睡得特别早，但是我在初一的时候起床时间是五点，然后看一会书再去上学，那个学期我进步就很大。高一每天浑浑噩噩，但是倒是可以拿着进步奖进而拿优秀奖，那个学期我也是非常之努力，但也有仪式感在里面，比如早读结束大家都去吃饭而我选择做篇阅读再出发。大学里唯一几次努力学技术，就是规定自己每天必须到实验室去学习，包括读书速度最快那一阵，每周五晚上都去图书馆呆着。

所以是否严格和辛苦，其实是影响不大的。关键要给自己的大脑一个信号做准备：要开始做一些辛苦的事情了。而目前看，晚睡对我来说是不会有这种仪式感的了，毕竟呆在宿舍，那么就唯有早点起床。包括我现在敲得这些废话，都是因为起的早，大家都在睡觉，我这才能想到什么写什么，无所顾忌。 7:30

六月

六月过的很快，转眼就七月一号了，上个月计划完成两件事，额，完成一半吧，也就是完成其中一件2333，看coursera的计划一拖再拖，最后就没办成。本来还想着前半个月完成第一件–毕设，后半个月上几天coursera绰绰有余来着。谁知道一个论文改来改去，我越是心急着想毕其功于一役，越是不行啊，感觉每次把论文发给导师，导师都像全然不记得上次他交代需要改的地方了，于是每次发都能看到新问题。最后就是16号检查完程序，23号答完辩，27号成绩都有了，28号才让我们去交论文最终版，而29号都是毕业典礼了。老师规格严格功夫到家的要求真是让人可敬可恨。如此一来非但没有因为临近毕业而闲起来，改论文之间还有各种事情需要去办：聚餐，喝酒，拍照，盖章，申请，整理。。。就很烦，混乱属性不可控制的疯狂扩撒，都没空感神伤怀。一直觉得应该给大学记录点什么，索性放在这一起写了吧。懒得很。

其实我早就打算写点什么，主要是前段时间找到了高三毕业时老师逼着我们写的总结，虽然写规划的那篇找不到了，虽然总结写的也是无比矫情–可能那时候可能还受着高考的荼毒，恨不得引天引地引空气，抽空还在那儿琢磨着修辞文采和半通顺的长句子。但是回顾起来：

反思

想来大一参加社团其实收获寥寥，因为目标不明确，后来意识到自己的社交技能树以前没点过，所以就都退了。迷茫这个词年轻的时候可以任性的用一用，大二，大三，大四总不好再用这种借口开脱。而且轻言放弃是个很蠢的事情，从最后的结果上看，坚持不断学习的发哥不出意外的成功保研了，而我可不记得他当时成绩比我好很多。大二确定毕业就去工作，为了这个做了一点点准备，也确实目的清晰的做了那么一点微小的工作，但是做得并不好。我这人有种得过且过的懒人心态，再不像小时候那么争强好胜，可是实际上在小的团队里，成绩中上的混子是没有出头之日的。以后工作也是，小团队里做到最好才有价值，我要切记这一点。大三实习，非要玩漂移，既定路线走的不舒服，回头就想换方向，可是哪有那么容易，考研准备的时间不充分，还记得一夫在我三月份找实习的时候就开始准备复习了，而我九月接近十月才开始专心复习。蜜汁自信的最后结果却是求而不得，而且还是考试这种形式。除了时间不充分以外，重点不突出，而且给自己定的目标也不够高，就想着过了分数线就行。。。我确实过了往年分数线，谁知道他会提升啊。好在最后结果也不算不能接受，换个环境对我来说应该是更好了。

矫情

我说感谢室友不杀之恩固然是玩笑之语，但我的感情是认真的。虽然从没和谁发生什么矛盾争执，但我总觉得自己干了不少蠢事。因为我深知自己情商很低，除了极少情况下我索性装傻以外，大部分时候我是真傻的。必须感谢各位对我的包容。鸫哥性子很直的，也讲给我很多道理，虽然我迷迷糊糊的，还是希望能学到那么一点点，也希望鸫哥效率依旧，毕竟在我看来，鸫哥要是真心想做成一件事情，那就没什么能阻挡的住，一定要和鸫嫂幸福；发哥是真的很励志的，希望我们开黑的时候没对你造成太多干扰，也希望发哥一如既往的认真学习，不说功成名就，回报与收获总是会有的，女朋友也是会有的，而且是漂亮的。主席生活多姿多彩的很，让我这个宅属性的凡人见识了很多新操作，可是主席的能力我羡慕不来，惟愿主席立业成家一切顺利，尤其是和学姐照顾好两只金毛。而微微，冠神，虽然不知道为什么，总是觉得很高端的样子，只希望微微记得出国之后，苟富贵，勿相忘啊。隆隆，性子稳的很，有个什么事情我还是觉得拜托你很靠谱，倒是希望你和发哥学习一下，稍稍努力一点，把肚皮减一减。而天泽又高又富又帅，不嫌我菜，不嫌我贫，一时间还想不到能祝人生赢家什么，就祝你永远年轻，永远是个宝宝吧。至于翔哥，和你赌了一顿饭，虽然你总是骗我，但我真希望最后是我请你，也愿你接下来的学习和工作一切顺利。

回家

这两天回到家，身心放松睡的深沉，每次都感觉是从万丈深渊里醒来。下午收拾了一下自己黑历史，除了各种成绩单证书和照片，甚至还有小升初考完试的日记，想想也是感谢老妈。里面甚至还有高中读书每次往家里打电话之后老妈的记录，我根本不敢看，我之所以没有顺着懒散的性子堕落下去，除了机缘巧合活在一个强者如云的环境里，老妈的关心和帮助居功至伟。我其实对婚姻和孩子很恐惧，因为我知道我根本不可能像父母爱我一样爱我的孩子更不可能欺骗自己去爱自己以外的其他人。昨天去见了一下奶奶，只觉得生命脆弱，时间无情，以前总是想什么人生意义，老爸这两天倒是不断跟我分享什么是幸福：身体健康，钱够花。可我只觉得自己可怜，从不会为自己的经历热泪盈眶，我在这回顾过去也是冷静的很，往往变成了批评与自我批评，偶有感时伤怀也被自己定义为矫情。

但愿

新的旅程，我但愿自己能在选定的方向上坚持不懈地付出努力，顺便找到一个好奇不减的爱好就更好了。再大一点空一点，我但愿这世界仍是和平的主旋律，种族宗教和分裂问题不要发生在身边。我但愿中国这谜一般坚挺的房价和gdp能依旧坚挺，阶级固化不要那么快，让我们还有一丝实现阶级跨越的希望。我但愿我认识和认识我的人长命百岁，倒不是为了苟活于世，只为了见识这世界飞速发展带来的无限可能，若是没有老的糊涂，顺便看看当年身在山中不知始终的选择究竟路向何方就更好了，虽是不曾指点江山挥斥方遒，但总可以看看同一时代到底几人射雕几人比高，鼓个掌也是好的。

我发此宏愿，若是天才人物说不定就天地共鸣，异象纷程了，而我看了一眼窗外，一片漆黑，十一点了，也不知道敲了多少错别字，睡觉睡觉。

def simple(request):
    import random
    import django
    import datetime

    from matplotlib.backends.backend_agg import FigureCanvasAgg as FigureCanvas
    from matplotlib.figure import Figure
    from matplotlib.dates import DateFormatter

    fig=Figure()
    ax=fig.add_subplot(111)
    x=[]
    y=[]
    now=datetime.datetime.now()
    delta=datetime.timedelta(days=1)
    for i in range(10):
        x.append(now)
        now+=delta
        y.append(random.randint(0, 1000))
    ax.plot_date(x, y, '-')
    ax.xaxis.set_major_formatter(DateFormatter('%Y-%m-%d'))
    fig.autofmt_xdate()
    canvas=FigureCanvas(fig)
    response=django.http.HttpResponse(content_type='image/png')
    canvas.print_png(response)
    return response
