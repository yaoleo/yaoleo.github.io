六月

发表于 2016-07-02 | 分类于 画家 |

自己不能感动自己

这个月，考试月，非常忙。哈尔滨凌晨三点就天亮我不知见了几次，甚至习以为常。

最忙的时候白天电子实习，晚上考试，还要通宵复习。好在基本都通过了，成绩也马马虎虎。起码考试的成绩还行，只是课程作业和实验以前也没认真对待过。不过我倒是不在意这些。

但我还是很清楚即使这不是咎由自取，我也没有理由自己感动自己。我之所以这么忙，甚至是有点辛苦，并不是我的努力。有趣的是这大部分是因为我之前并没有在学业上付出太多。以至于面临考试的时候只能手忙脚乱抱抱佛脚。另一部分原因才是偶然与巧合的事情安排冲突到了一起。

但我觉还好，也并不是太辛苦。唯一不好的是可能因为缺少睡眠，那几天总是丢三落四，不是忘钥匙，就是忘学生证，要不就是忘作业。可见熬夜对我来说还是件不太好的事情。对我的健康有很不好的影响。也很影响我的效率，要尽量避免这样的生活。

当然，熬过那几天，其实我也睡不了太久了。如果我十二点之前睡，那么早上六点之前就会醒。虽然白天我还是会困，但我基本上不能连续睡六个小时以上了。不知道这是不是一个好的生物钟。

长大

国华这个月返校庆典，主题就是“我们长大啦”。没机会回去，非常非常遗憾。想想真的很快，我们已经大三了，除了学长们在朋友圈发毕业的消息外，同届的同学很多都在发大学最后一门考试，大学最后一门必修课之类的。大四我们已经在各自的方向上分道扬镳，时间会过得更加快吧，而毕业真的已经近在眼前。

老罗来哈尔滨实习，恰好我的小学期还没开始，花了两天时间陪玩，聊点陈年旧事，聊点未来规划，其实挺好的。让我非常还念那些老同学。每个总是回忆的人就肯定已经老了，不是在身体上就是在精神上。那天遇到一个孩子，兴致勃勃的拉着父亲喊着天鹅，而我却只看见了鸭子。这就是长大的代价吧。

租房

租房子是件有意思的事情。当时手写了一篇记录，非常有趣，虽然字还是丑。

遗落

忙是真忙，所以也有很多事情遗落了。把注意力集中在应付课程考试上。

建筑画，单词，跑步，听力。

我得重新捡起来继续坚持。

new life

下个月需要先上半个月的苦逼课，非常忙。毕竟要修七门课程。 我只能尽量吧。

而且这半个月还要把backbone学好。总不能什么都不会就去实习吧。

接下来就要开始实习生活了。体验一下，也不知道究竟是什么样的，有点期待吧，当然更多的是对未来的担心。不过倒是没关系，我想要的太多，我自己也清楚这一点。但是我会告诉自己不要着急。 聊胜于无吧。

五月

发表于 2016-05-27 | 分类于 画家 |

陈述

想了半天，没想出来这个月的主题。明天就是无聊的运动会啦，想起去年运动会前一天，我还在实验室通宵做项目然后早上直接回宿舍门口集合去运动场，而现在只能无所事事的在这敲点汉字。却还想不出来写点什么。周五，lol无限火力，明天运动会放假，也没什么要交的作业。舍友都在宿舍开黑。可我就是不想玩。我也不知道，为什么有一点点失落感。总之是心里空空的感觉。

时间飞快，是真的。我都想不到今天就已经要写第三篇这个类似月记的东西啦。话说从小到大，交过日记周记，还真是只有给自己写才能写出月记这种不伦不类的东西吧。

照例先陈述一下所谓得失吧。

第一次完成了一整张的建筑画，虽然丑的很，还是在脑机课堂上画的，不过还是不错不错。然后开始用abomoo做听写，今天才是第二天，不过感觉不错。至于早起，今天倒是勉强完成了，但是上课很困，生物钟完全没有调整过来。知乎答题抄了一句王小波骗了接近百赞，而自己认真写的却收获寥寥，其中关于健身app的盈利问题赢得了旅客app创始人的认同，借机会微信和他聊了几句，但其实没太大的收获。读了两个半本的佛学书，《正见》，《说佛》。建了静态博客yaoleo.github.io。ife任务彻底结束，虽然我第四阶段没做什么有价值的工作。和携程HR交流把实习时间推迟到7月18号。这个倒是不不错，希望自己能把学分修完吧。

最大的失大概从上面也可以看出来，没错，在技术上我觉得自己没什么收获。当然有这种感觉的原因可能是因为我觉得上课的那些东西，都不能算是我在技术上的收获。可是我却不得不完成无聊的实验和实验报告。另外，可怕的是我中断背单词将近两周，这种坚持的事情对我来说从来都是只要中断就会放弃的。但是这次还好，因为我还有记录，我既然已经记得这一点，我一定会继续坚持下去。跑步这件事虽然有下雨的借口，但我也没坚持好。给自己两个差评。

佛学初窥

这种哲学的东西还是可以抽时间读一点，但接下来千万不要读了，我要把注意力放在技术上。有一点思考，也把他复制过来一下。

可能因为所受的教育本身不支持宗教，反正我原先以为宗教和迷信是正相关的。但是初步读了一点书却发现并非如此，有很多的是偏见。我很高兴自己能认识到自己的无知或者说是误解。

然而我竟然带着『我对佛学的了解很少，很多佛学思想都是对的，只是我理解有错』的想法去进一步读书。

昨天周末写总结的时候，突然发现这真是一件可怕的事情。

这样的结果就是我会默认佛学是正确的，然后认为自己阅历不够，没有找到真正能说服自己解释或者自己还没有理解到位。于是我会不断在佛学中，找到自己能够理解并支持的说法说服自己。

简而言之，就是即使我本身不这样想，我也会在浩瀚的典籍中找到自己满意的解释为止。

于是我相信的是什么?_?
我实际还是相信自己。
只不过我需要从佛学中找到支持自己依据。进而会误认为自己相信的是佛学。而真正信佛的人，估计不会也不应当如此。

本来计划看一点经书，而当我意识到这一点时，我想我暂时还是不适合看。所以大概我就是传说中的读书少想的还多吧。

反正我需要看点其他的换换思路。

凝视深渊不能太久。
同理，也不想被佛回以凝视，哈哈。

javascript数组需要注意的点

发表于 2016-05-22 | 分类于 黑客 |

Array.prototype.map()

map() 方法返回一个由原数组中的每个元素调用一个指定方法后的返回值组成的新数组。

map 方法会给原数组中的每个元素都按顺序调用一次 callback 函数。callback 每次执行后的返回值组合起来形成一个新数组。 callback 函数只会在有值的索引上被调用；那些从来没被赋过值或者使用 delete 删除的索引则不会被调用。

callback 函数会被自动传入三个参数：数组元素，元素索引，原数组本身。

如果 thisArg 参数有值，则每次 callback 函数被调用的时候，this 都会指向 thisArg 参数上的这个对象。如果省略了 thisArg 参数,或者赋值为 null 或 undefined，则 this 指向全局对象 。

##Cookie, LocalStorage, SessionStorage, Session

Web Storage和cookie

Web Storage包括了两种存储方式：sessionStorage和localStorage

sessionStorage用于本地存储一个会话（session）中的数据，这些数据只有在同一个会话中的页面才能访问并且当会话结束后数据也随之销毁。因此sessionStorage不是一种持久化的本地存储，仅仅是会话级别的存储。

而localStorage用于持久化的本地存储，除非主动删除数据，否则数据是永远不会过期的。
